About

This is a demo app that showcases displaying Fitbit data.

Features:

  • Login using Fitbit account via OAuth 2.0
  • Show daily steps and calorie summaries for the last 7 days
  • Show daily steps and calorie summaries for the last 30 days

Under the Hood:

  • Python backend using Flask
  • Bootstrap frontend
  • D3 visualization
  • Functionality to make all Fitbit API calls

Credits:

  • rauth: https://rauth.readthedocs.org
  • OAuth 2.0 tutorial: http://blog.miguelgrinberg.com/post/oauth-authentication-with-flask
  • Flask: http://flask.pocoo.org/
  • Bootstrap: http://getbootstrap.com/
  • D3: https://d3js.org
  • Python class that wraps Fitbit API calls is adjusted based on this project: https://github.com/orcasgit/python-fitbit
